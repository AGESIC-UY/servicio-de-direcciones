Ejemplo de barra de búsqueda

Barra de navegación con búsqueda de calles de la IDE de Uruguay.

Para ver cómo se comporta en varios tamaños, cambia el tamaño de la ventana.

El API de búsqueda está en API Direcciones UY .

Las llamadas que se usan en el componente son 3:

  • /api/v1/geocode/candidates => Cuando tecleas, se hace esta llamada.
  • /api/v1/geocode/find => Cuando el usuario selecciona la entrada, se llama a esta función
  • /api/v1/geocode/reverse => Si vas a emplear un mapa, esta función obtiene la dirección más cercana a una posición
